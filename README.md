# User Management API

This is a professional RESTful API built with ASP.NET Core for managing user data.

## Features
- **Full CRUD Support**: Manage users with GET, POST, PUT, and DELETE endpoints.
- **Data Validation**: Strict input validation using Data Annotations.
- **Custom Middleware**: Integrated logging middleware for request tracking.

## Development Note
> **Developed and debugged using GitHub Copilot.** > Copilot assisted in generating the CRUD boilerplate, refactoring the middleware, and fixing model validation issues.
